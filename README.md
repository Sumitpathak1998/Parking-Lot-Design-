Project Overview

The Parking Lot Management System is a console-based application developed using Python and Object-Oriented Programming (OOP) principles.
The system manages parking operations such as:

Parking vehicle entry
Vehicle exit and charge calculation
Parking floor management
Parking spot management
Parking attendant management
Parking rate management

The application follows a role-based operation system where different users have different responsibilities.

Roles and Responsibilities

Admin : 

1. Create Parking Lot
2. Add Parking Floor
3. Remove Parking Floor
4. Add Parking Spot
5. Remove Parking Spot
6. Add Parking Attendant
7. Remove Parking Attendant
8. Add Floor Panel (entry/exit) for current case each floor has only one entry and exit gate
9. Remove Floor Panel
10. Modify Parking Rate

Parking attendant 

The Parking Attendant handles daily parking operations.

1. Vehicle Entry Operation
	- Register incoming vehicles
	- Allocate parking spots
	- Generate parking tickets

2. Vehicle Exit Operation

Parking attendant can:

	- Scan parking ticket
	- Calculate parking charges
	- Collect payment
	- Update ticket status
	- Mark parking spot as available

Exit process includes:

	- Exit time calculation
	- Parking duration calculation
	- Charge calculation
	- Payment verification
